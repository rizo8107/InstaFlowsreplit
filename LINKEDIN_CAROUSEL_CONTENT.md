## Design Specifications

### Color Palette
- Background: #2d1b4e (deep purple)
- Gradient End: #4a2c6f (lighter purple)
- Primary Text: #e8dff5 (light lavender)
- Accent: #a78bfa (bright purple)
- Highlights: #ffffff (white)

### Typography
- Heading Font: Inter Black or Montserrat Bold
- Body Font: Inter Regular
- Heading Size: 48-64px
- Body Size: 18-24px

### Layout Guidelines
- Profile photo: 60x60px, top left
- Margins: 40px all around
- Card radius: 20px
- Consistent spacing: 24px between elements
- "Swipe â†’" indicator: bottom right, 16px

### Visual Elements
- Rounded cards for text blocks
- Subtle glow effects on headings
- Clean icons (outline style)
- Minimalist, professional aesthetic
- High contrast for readability

## Tips for Creating in Figma/Canva

1. **Use the reference images** for exact color matching
2. **Keep it minimal** - don't overcrowd slides
3. **Use consistent spacing** throughout all slides
4. **Add subtle animations** if posting as video
5. **Test mobile view** - most LinkedIn users on mobile
6. **Export at 1080x1080** for best quality
7. **Add "Swipe â†’"** indicator on every slide except last
